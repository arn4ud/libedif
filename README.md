# libedif

An edif parser and driver.

## Getting Started

```
git clone https://github.com/arn4ud/libedif.git
cd libedif           # Main directory of the cloned repository.
mkdir build          # Create a directory to hold the build output.
cd build
cmake ..             # Generate makefiles
make
```
